# AI Workout Tracker

Simple Python app that:
- Converts natural language workout input to structured exercise data using Nutritionix.
- Logs each exercise to Google Sheets via Sheety.

## GitHub Description
A simple Python workout tracker that parses exercise text with Nutritionix and stores results in Google Sheets via Sheety.

## Setup
1. Create and activate a virtual environment.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Copy `.env.example` values into your environment (or `.env` if you use a loader).
4. Run:
   ```bash
   python main.py
   ```
